# AI 主池打板 · 量比日内策略(v2.1)

AI 主线 60 只票、集合竞价量比驱动的 **T 开盘买 / T+1 收盘卖**、两等份错开一天滚动的日内策略。含完整指令文档、回测引擎与逐年报表生成。

## 文件
| 文件 | 说明 |
|---|---|
| [`当日选股指令_v2.1.md`](当日选股指令_v2.1.md) | **实盘当日选股指令**(择时、硬过滤、主选/兜底/深兜底、个股盘口过滤、卖出规则) |
| [`回溯指令_2021-2026_v2.1.md`](回溯指令_2021-2026_v2.1.md) | **历史回溯指令**(逐日重放口径 + 每年一表的 Excel 输出规范) |
| `fetch.py` | 用 Tushare 拉 60 只票的日线/前复权/集合竞价 → `rawdata.pkl` |
| `engine.py` | 回测引擎,逐日重放,输出 canonical 结果 `res_gated.pkl` |
| `excel_year.py` | 生成每年一个工作表的美化 Excel(逐年收益率 + 净值曲线) |
| `requirements.txt` | 依赖 |

## 策略要点(v2.1)
- **择时**:60 只等权指数在 MA20 上方→进攻;下方→防守(空仓)。
- **全局硬过滤**:开盘>9.8% / 20日涨幅>100% / 量比>20 全档剔除;**阴线高点距>7天**剔除;**个股高开不买/低开不买**(见指令)。
- **主选**:量比∈(3,20)、贴高(≥60日高×93%)、多头排列、成交额(T-1)≥1亿、非ST;选1/2/3 按量比名次定位、不顺延,选2/3 须 20日涨幅≤80% 且开盘>−1.5%。
- **兜底 / 深兜底(量比>5)**:逐级放宽。
- **卖出**:T+1 收盘卖;**T 日跌停的笔改 T+1 开盘卖**(风控)。

## 回测结果(2021-01 ~ 2026-06,canonical)
期末净值 **2420×**、最大回撤 **−24.2%**;逐年 +148% / +88% / +248% / +62% / +757% / +969%。

> ⚠️ **重要保留**:股票池为当下 AI 主线 = **幸存者偏差**;回测**零摩擦**(未计佣金/印花税/竞价涨停买不进);部分个股规则系单票历史统计、**过拟合风险高**。**绝对收益严重高估、不代表实盘**,应主要看相对结构与分年差异。

## 用法
```bash
export TUSHARE_TOKEN=你的token
pip install -r requirements.txt
python3 fetch.py          # 拉数 → rawdata.pkl
python3 engine.py         # 回测 → res_gated.pkl(打印逐年收益)
python3 excel_year.py     # 出 Excel 报表
```
