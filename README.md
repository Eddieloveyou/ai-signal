# AI主池·量比策略 v2.4 — 每日早盘信号

云端 routine 每交易日 9:25 克隆本仓库并运行 `daily_signal.py`，输出**当天买哪几只**并推送到你的飞书/企业微信/Telegram。

## v2.4 两处实证修正(相对 v2.3)
1. **根除未来函数**：量比与开盘价改用 `stk_auction`(纯9:25撮合: `vol`股 / `price`撮合价),**不再用 `stk_auction_o`**——后者混入9:30开盘后成交,属未来函数(你9:30才买,那批量当时不存在),会把量比虚高、把次级票误抬进主选(实测江丰电子/长川科技曾因此假入选)。
2. **低开过滤替掉终值位置**：终值位置需9:15–9:25完整路径,Tushare取不到(stk_auction无高低、auction_o是未来数据)。改用**点位时间干净的低开比例**过滤 #2/#3。

## 规则 v2.4
- **择时**：60只等权指数 T-1收盘 > MA20 → **进攻(在场)**；≤ MA20 → **防守=空仓**(持现金,收益0)。银行/黄金ETF仅参考不持有。
- **量比** = (今日9:25**撮合量**/100) ÷ (过去5日日均量/240)。开盘价=撮合价。
- **候选基础筛选**：量比∈(3,20) + 贴高(T-1收盘≥60日高×93%) + 多头(收盘>MA20>MA60) + 近5日均额≥1亿 + 开盘涨幅<9.8% + 非ST。
- **主选(最多3只, 均分仓位)**：
  - **主选一**：候选中剔20日涨幅>100%后, 量比第1 —— **低开照买**(#1龙头低开是优质买点)。
  - **主选二/三**：剔20日涨幅>80%后取量比次高的自然 #2/#3, 各自**仅当开盘涨幅 gap ≥ −1.5%(低开不超1.5%)才保留**;低开超阈值 → **该格留空、不替补**(替补会买到低量比杂票,回测更差)。
  - 低开容忍 `LOW_OPEN_TOL=-1.5%`:扫描显示它跨2024(震荡)/2025–26(牛市)两段最稳健、回撤最低。代码里一行可调。
- **兜底**(主选0只)：去掉量比∈(3,20)与20涨限制(留贴高+多头+额≥1亿+开盘<9.8%+非ST) → 量比第1满仓。
- **深兜底**(主选+兜底0只)：仅量比<20+近5日均额≥1亿+开盘<9.8%+非ST → 量比第1满仓。
- **执行**：T开盘买、T+1收盘卖,本金两份错开一天滚动。
- ⚠ 量比历史不可回溯(stk_auction仅当日/近日);老日期回测会回退auction_o并打标,结果仅供参考。回测均含幸存者偏差+AI牛市样本,绝对收益虚高。

## 用法
```bash
export TUSHARE_TOKEN=你的tushare_token
# 推送任选其一(配了就推):
export LARK_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxxx   # 飞书自定义机器人
export WECOM_WEBHOOK=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=xxxx  # 企业微信群机器人
export TELEGRAM_BOT_TOKEN=xxxx; export TELEGRAM_CHAT_ID=xxxx            # Telegram
export PUSHPLUS_TOKEN=xxxx                                             # PushPlus(个人微信)
python3 daily_signal.py            # 取今天; 9:25后运行出当日买入
python3 daily_signal.py 20260626   # 指定日期(回看/测试)
```
仅输出当天买入标的，不输出收益率。详见 strategy_spec.txt。

# AI 主池打板回溯引擎 v2.1 — `backtest.py`

`daily_signal.py` 只出**当天买哪几只**；`backtest.py` 则**逐日重放 2021–2026 全程**，复算量比与关键指标、套用全局硬过滤与主选/兜底/深兜底选股、计算两份错开滚动净值与逐年收益率，并导出**统一美化的 Excel 工作簿(每年一个工作表)**。规则依据见 `backtest_spec.txt`。

## 与日内 v2.4 的口径差异
- **数据源**：回测唯一可得的竞价源是 `stk_auction_o`(9:15–9:25 开高低收量，历史可回溯)；`stk_auction` 纯撮合接口历史取不到，仅当日可用。竞价最终价 = `open` = 当日开盘价。
- **前复权全程一致**：`daily × adj_factor`，以区间内最新因子为基准。
- **选股用名单化硬过滤**(高开不买 / 低开不买名单)而非单一低开容忍；终值位置仅展示、不参与选股。

## 规则要点(完整见 backtest_spec.txt)
- **择时**：60 只等权指数(日收益等权) T-1 收盘 > MA20 → 进攻；否则防守=空仓(收益 0)。
- **量比** = (竞价量股/100) ÷ (过去5日日均量手/240)。
- **全局硬过滤**(任一命中整票剔除)：开盘>9.8% / 20涨>100% / 量比>20 / 60日最高K为阴线且距今>7日 / 高开不买名单(gap≥0) / 低开不买名单(gap<0)。
- **进攻候选**：量比∈(3,18)+贴高(≥60日高×93%)+多头(收盘>MA20>MA60)+成交额(T-1)≥1亿+开盘<9.8%+非ST。
  - **候选一**：量比第1，低开照买。
  - **候选二/三**：量比第2/3 名，须 20涨≤80% 且 开盘涨幅>−1.5%，否则该名次留空、不递补。
- **兜底**(主选0只)：去量比>3下限与20涨限制，留贴高+多头+额≥1亿+非ST → 量比第1满仓。
- **深兜底**(主选+兜底0只)：额≥1亿+开盘<9.8%+非ST+量比∈(5,18)+20涨≤100% → 量比第1满仓；无量比>5合格票则空仓。
- **净值**：本金两份错开一天滚动，T 开盘买、T+1 收盘卖；T 收盘跌停的笔改 T+1 开盘结算(主板≤−9.8%/双创≤−19.8%)；组合净值=两份之和(初始各0.5)。

## 用法
```bash
export TUSHARE_TOKEN=你的token
python3 backtest.py                                  # 回溯至最新交易日 → ai_backtest_2021_2026.xlsx
python3 backtest.py 20210101 20261231 out.xlsx       # 自定义区间与输出文件
```
控制台打印逐年收益率/累计收益/最大回撤；Excel 含「总览」表 + 每年工作表(逐日候选按收盘获利排名、近5日每分钟均量、买卖限制、入选档位、T开盘/T+1结算/收盘获利)。

> ⚠ 固定 60 池含幸存者偏差(后视镜选样)，账面历史收益仅供复盘找规律，不等于未来可复现收益。
